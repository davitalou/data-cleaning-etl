## Overview
Dự án này xây dựng quy trình tự động hóa xử lý dữ liệu từ file Excel và cập nhật dữ liệu vào Oracle Database. Dữ liệu được làm sạch bằng Python trước khi được đưa vào cơ sở dữ liệu để phục vụ phân tích và báo cáo.

## Main Features
- Đọc dữ liệu từ file Excel
- Làm sạch và chuẩn hóa dữ liệu bằng Python
- Kiểm tra dữ liệu thiếu, trùng lặp và sai định dạng
- Kết nối Oracle Database
- Tự động cập nhật dữ liệu vào bảng Oracle
- Hỗ trợ xử lý dữ liệu cho báo cáo và phân tích

## Workflow
1. Đọc dữ liệu từ file Excel
2. Kiểm tra và làm sạch dữ liệu
3. Chuẩn hóa tên cột và kiểu dữ liệu
4. Kết nối tới Oracle Database
5. Ghi dữ liệu đã xử lý vào bảng Oracle
6. Kiểm tra kết quả bằng SQL Developer

## Result
Dữ liệu từ Excel được làm sạch và tự động cập nhật vào Oracle Database, giúp giảm thao tác thủ công và tăng độ chính xác trong quá trình xử lý dữ liệu.
